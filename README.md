### 내가 맡은 역할 : 클라이언트 - 서버 - 드론간의 실시간 통신 설계

1. 사용자가 서버에 드론 요청을 한다.
2. 서버와 드론은 멀티스레드 기반으로 실시간 소켓통신을 연결한다.
3. 드론으로부터 오는 실시간 GPS좌표 정보를 클라이언트에 전송하고 드론의 실시간 위치를 지도화면에 표시한다.
